Razviti klase: 
- MediaFile (apstraktna):
  - string naziv
  - int trajanje
  - virtual void Play()
- AudioFile (nasljedjuje MediaFile):
  - int bitrate
- VideoFile (nasljedjuje MediaFile):
  -int framerate
- Media (surogatska klasa za MediaFile):
  - MediaFile* file
  - void Play()
Klase AudioFile i VideoFile definisu svoje Play metode.
Dodajte adekvatne konstruktore za sve klase. (trebace vam i DajKopiju() metoda)
Klasa Media ima konstruktor koji prima konstantnu referencu na MediaFile
Neka Play() metoda samo ispisuje atribute date klase.
